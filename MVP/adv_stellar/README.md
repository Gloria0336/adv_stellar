# 星際漂流 · adv_stellar

跨平台、由 LLM 擔任遊戲引擎的太空文字冒險。遊戲資料即此 repo。

## 怎麼玩
把 `PROTOCOL.md` 交給任一支援讀取本 repo 的 LLM（Claude 可直接透過 GitHub 連接器），
它會依規則載入 `state/` 存檔與 `data/` 規則開始遊玩。無連線的平台（如 Grok）可手動貼上
`state/core.json` + `state/ship.json` 續玩。

## 結構
- `PROTOCOL.md` — LLM 引擎規則（含防飄移正典規則）
- `GAME_DESIGN.md` — 完整設計藍圖 v1.0
- `state/` — 玩家存檔（會變動）
- `data/` — 靜態規則（唯讀；含 `data/canon/` 不可違背正典）
- `session/` — 場景脈絡與長期摘要

## 核心特色
- 飛船＝基地＝角色之家＝勝利條件；修復即擴充，甲板有地形。
- 雙軌成長：技能星網（異能）× 科技四大類（飛船）；雙樹交叉依賴。
- 母艦戰鬥：裂潮蜂群間歇來襲，可守可反擊。
- 判定：2d6＋細修（10+/7–9/6−）。
- 三層任務與深層變異三選一結局。

> 設計仍在演進；本為 P0 起始內容，技能/科技/模塊/星球皆為獨立檔案，新增不需改架構。
