## 0. 一句話定位（Steam 短描述草稿）

> 在一艘被未知災潮重創的巨型探索船上醒來，一趟趟潛入異星地表採集求生、
> 拼裝復活你的母船、與排斥你的星系文明周旋——最終決定如何面對那股撕裂星際的力量。

**類型標籤**：Roguelite・基地建造・俯視動作・科幻探索・extraction
**對標**：《死細胞》的動作循環 × 《缺氧/環世界》的基地拼裝 × 深岩銀河/Lethal Company 的「採集撤離」張力 × 《黑帝斯》的累積敘事。

## 1. 設計支柱（不可動搖的四根柱）

1. **飛船三位一體**：母船 ＝ 永久基地 ＝ 角色之家 ＝ 勝利條件。修復即擴充。
2. **兩層成長結構**：永久 meta（母船、科技、技能星網）＋ 每趟 run 臨時 build（裝備、拾取增益）。
3. **採集—撤離張力**：每趟遠征都是「背包容量 × 氧氣時限 × 撤離點」的風險取捨。
4. **天外即惡的孤獨感**：你是星系眼中的「天外之威脅」，整個遊戲是「證明自己不是威脅」與「對抗那股力量」的長期工程。

## 3. 世界觀（沿用原正典，摘要）

> 完整世界觀、KNOWN/HIDDEN 正典邊界與「裂潮」災變正典，**全數沿用** `MVP/adv_stellar/GAME_DESIGN.md` §1、§14。
> 本節僅摘核心，正式版的劇情演出須嚴守原正典的 KNOWN/HIDDEN 揭露閘門（見 §10.3）。

- **前提**：你的種族派大型探索船進入未知星系（官方任務＝探索科研；真實目的＝取得「那股星系級力量」與掌握法，**開局主角不知情**）。
- **災變**：進場前遭蜂群式災潮**「裂潮」**重創，27 死，僅存你＋休眠中的**通訊官**與**科學家**。船失動力漂流數日後被一顆星球引力鎖定於軌道（＝起始世界）。
- **裂潮**：某星系級力量的衍生物（半實體異象實體），間歇來襲、可守可反擊、智能不升級。
- **天外即惡**：星系文明相信「傳送、資源、生命皆源於行星本身，故星外皆惡」——這解釋原住民的排斥、好戰種族的敵意、傳送機制對你的排斥。
- **行星傳送**：原住民靠星系力量的「馴化版」傳送，與好戰種族的「武器版」同源。你初期被排斥 → **相容性**成長線逐步解鎖更多星球。
- **控制中樞 AI**：受損修復＋夥伴型常駐 AI，隨母船修復逐漸恢復人格，是孤獨虛空中的羈絆與情感支線。

## 5. 母船建造系統（META 永久層・拼圖式）

### 5.1 不規則殘骸網格（修復即擴充）

母船甲板是一張**繼承自殘骸的網格**，非空白棋盤：

- 大部分格子**鎖死**（報廢艙段），用「結構・材料」科技逐塊清理解鎖。
- **解鎖格子 ＝ 修復進度 ＝ 勝利條件進度線**（永久、只增）。
- 理論最大值＝可容納約 200 人的完整巨艦（終局超大基地）。
- 修復路徑須**從仍運作艙段向外接力**，不能跳修孤立遠格 → 擴張有地理與順序。

### 5.2 格位地形（tile traits・沿用原五種）

| 地形 | 適合放置 | 效果／代價 |
|---|---|---|
| 船脊・動力幹線 | 反應爐、高耗能模塊 | 出力/連線加成；稀少，兵家必爭 |
| 船殼緣（對外） | 砲塔、感測、機庫、氣閘 | 解鎖對外功能；裂潮防衛戰時最先被打 |
| 核心內艙（受保護） | 休眠艙、研究、指揮 | 防裂潮、無對外通道 |
| 破損區 | 任意（便宜快速） | 不穩定，需先穩固，否則觸發災害事件 |
| 冷卻區 | 熱源、反應爐 | 抵銷輻射/熱擴散；位置固定 |

### 5.3 拼圖式擺放（已定版）

- **模塊形狀**：矩形 **1×1 / 1×2 / 2×2 / 2×3**（不採 L/T 等不規則形，控制製作與平衡複雜度）。
- 大模塊＝大形狀＝更難塞進不規則殘骸 → **「怎麼塞得下、塞在哪」就是策略**。
- 形狀同時對上**地形**才拿滿地形加成。
- **編輯時機**：**僅在母船整備階段**可自由重排（出航中、防衛戰中原則上鎖定；防衛戰的緊急應變屬另議的擴充，預設不開放即時改建）。

### 5.4 能源 / 電力網（已定版）

- **需管線/電力網連接**：模塊必須實際連到電力幹線才運轉（非僅看全船總量）。
- 反應爐（須置於船脊/動力幹線）為電力源，向外鋪設電力網/管線。
- 佈線本身是一層謎題：距離、繞線、瓶頸都要規劃。
- **三重擴充上限**：①格數（修復進度）②能源（電力網收支）③核心模塊前置（大模塊需特定核心）。

### 5.5 相鄰 Combo（沿用原三層）

1. **相鄰加成**：相鄰特定模塊互增益（如 能源↔生產 +15%）。
2. **組合解鎖**：湊齊一組解鎖新功能（如 反應爐＋異象導管＋共鳴核心 → 全船能源 +40%）。
3. **互斥干擾**：某些不能相鄰（如 反應爐↔居住 → 船員輻射 -20%）。

> 兩層 placement 策略：地形（§5.2）× 相鄰 combo（§5.5），再疊電力網佈線（§5.4）。

### 5.6 資源經濟（v3・已定版・跨系統）

> 資源是模塊「產出/消耗」的基礎，跨母船建造、遠征採集、科技研究三大系統。分三層：

**A. 即時系統值**（不採集、不囤積，只看當下收支）

| 值 | 作用 | 來源／限制 |
|---|---|---|
| ⚡ 電力 Flux | 電力網**即時收支**，模塊運轉 | 反應爐產、模塊耗；**不可囤積** |
| 👥 居住容量 | 船員數上限 | 生命維持/居住模塊決定 |

**B. 物理資源**（採集或生產 → 入母船庫存；遠征中死亡則**損失未撤離部分**）

| 資源 | 主要用途 |
|---|---|
| 🔩 合金 | 建造/修復模塊、解鎖格子 |
| 🌌 異象結晶 | 變異/異能解析、特殊模塊 |
| ⛽ 燃料 | 母船自航（§6.5） |
| 🔬 研究樣本 | 母船研究模塊解析 → 產科技點 |
| 🪨 星球特產 | 進階模塊/科技/外交；**每星 1~2 種，驅動跨星探索** |
| 💨 氧氣 | **可生產、可儲存**；出航灌瓶，遠征時當倒數計時消耗 |
| 🍞 糧食 | **船員維持**：船員隨時間消耗，不足則無法增員/效率下降；水耕・農圃模塊生產 |

**C. META 永久點數**（死亡**不損失**）

| 點數 | 用途 | 來源 |
|---|---|---|
| 🔧 科技點 | 科技樹解鎖 | 研究樣本解析產生 |
| ✨ 技能點 | 技能星網 | 里程碑/經驗/探索 |
| 🏆 GE 成果 | 中層任務目標（帶回母星） | 科研產出 |

**研究鏈**：遠征採 🔬樣本(B,可損失) → 母船研究模塊解析消耗 → 產 🔧科技點(C,永久) → 解鎖科技。
**醫療**（已定版）：**不設為資源**。母船醫療艙模塊治療（吃時間/居住）＋ 遠征 run 臨時拾取醫療包/治療異能（死亡即失）。

### 6.5 母船移動系統（新增・已定版）

母船有兩種移動狀態，是一條**獨立於「行星傳送」的 META 移動線**：

**① 前期・軌道地形系統（行星引力軌道內・首發核心）**
- 母船被起始星球引力鎖定，在引力軌道內**隨時間漂移**（非整備直選位置）。
- **防衛戰準備期**：看到當前位置的**防守地形（系統隨機生成）**——受襲方向、掩體、隱形危險區。
- **可花燃料 reroll** 隨機新地形；**後期研究解鎖「軌道圖」→ 直接挑選有利地形**。
- **位置/地形只影響防衛戰，不影響產出**（肥險解綁）。
- **詳細數值與對照見 §13-11。**

**② 後期・星際自航（修好主引擎/主動力後・EA 中期）**
- 解鎖後母船可在星球/區域間**物理自航**。**首發不做跨星自航**，留 EA 中期（§12）。
- **與行星傳送並存分工**：

  | 移動手段 | 用途 |
  |---|---|
  | 行星傳送（相容性線） | 快速往返**已解鎖**星球（靠好感/科研解鎖） |
  | 母船自航（主引擎） | 物理飛到**傳送到不了**的新區、**裂潮異象核心**、更好資源區位 |

- **代價**：**跨行星航行＝章節里程碑驅動**（非自由燃料飛）；軌道內地形 reroll＝固定燃料成本（§13-11）。
- **影響**：推進主線劇情（靠近裂潮核心等里程碑）、開啟更好的資源區位與遠征選擇。

> 母船移動屬 **META 永久層**決策，與 run 內遠征分離。軌道地形系統屬早期（首發）、星際自航屬中後期解鎖（EA 中期）。

## 7. 星球遠征玩法（RUN 臨時層）

### 7.1 關卡結構（已定版）
- **小區域自由探索 ＋ 區域推進**：一顆星球分若干可自由走動的小區域，清理/探索後推進更深區域。
- **前哨站建立**：可在星球上建立小型採集前哨（沿用原「星球前哨」），定期回收資源。**前哨經濟見 §13-8**（危險度影響＝被裂潮波及風險，非產率）。

### 7.2 戰鬥（已定版）
- **武器 ＋ 異能混合**：主武器（射擊/近戰）＋ 可裝配的**異能技能槽 2~4 個**（來自技能星網，§9.1）。
- 異能受 PSI/能量與冷卻限制；出航前於整備裝配技能槽。

### 7.3 事件呈現（已定版・取代原 2d6 文字事件）
- **重要事件**（劇情、關鍵抉擇、稀有遭遇）：彈出**文字事件＋選項**，依數值結算（保留原敘事豐度）。
- **雜事**（一般採集、險阻、機關、潛行）：**環境動作化**，直接操作互動物件，不彈文字。

### 7.4 採集與撤離（已定版・核心張力）
- **採集容量**：可帶回的資源/樣本受**背包 / 撤離倉容量**限制 → 帶什麼、丟什麼是即時決策。
- **氧氣限時**：遠征有氧氣（或等效資源）倒數，逼你控制停留時間。
- **撤離點限制**：須抵達**撤離點**才能撤離（非隨地撤）。
- **自由撤離時機**：在上述限制下，玩家自行決定何時撤——越深越肥但越險（貪婪張力）。
- **戰敗後果**：HP 歸零 ＝ **損失本趟全部攜帶**，空手回母船（標準 roguelite 懲罰）。

## 13-2. 數值框架（已定版）

> 全文最底層的數值基準，§5.7 模塊、§5.8 combo、§9.2 科技樹的具體數值都掛在此。
> 本節定**尺度基準與各子系統框架**；逐節點/逐模塊的細部數值待 §13-5（技能星網）與後續平衡。

### 13-2.A 尺度哲學（已定版）

- **精簡整數**：HP/PSI 個位~百位；資源/成本個位~千位；Flux 十位階。易讀、好平衡、適 solo 調數（《死細胞》風格）。
- **Flux 量化**：§5.7 的定性標記 → 數字。**消耗**：低 **10** / 中 **30** / 高 **50**。**產出**見 §13-2.E。
- **電力為即時收支**（§5.6，不可囤積）：看全船 Flux 淨值，模塊須連電力網（§5.4）才計入。

### 13-2.B 角色核心屬性（已定版）

| 屬性 | 基準 | 成長 / 規則 |
|---|---|---|
| **HP** | 起始 **100** | **保守成長至 ~200**（2 倍）；來源＝§9.1 蛻體柱・體能線 ＋ §9.2 生物強化科技。技巧>肉度。 |
| **PSI** | 基準 **100** | **脫戰恢復**：戰鬥中不自然回復，脫離戰鬥才回 → 逼玩家在單場戰鬥內精算 PSI 預算。 |
| 移速 / 基礎攻擊 | — | **待 §13-5**（手感數值，與動作原型一起調）。 |

### 13-2.C 氧氣系統（已定版）

- **計量**：抽象**氧氣點數**，基礎氧瓶 = **100 點**。
- **消耗**：**純時間扣點**（每秒固定扣，基礎約可撐 5 分鐘）。**環境危險（體溫/污染）獨立結算、不額外加速耗氧**（與 §13-4 一致）。
- 灌瓶上限/補氧裝置數值待平衡；製氧機（§5.7）產氧入庫供出航灌瓶（§5.6）。

### 13-2.D 背包 / 撤離倉容量（已定版）

- **計量＝重量上限**（非格子，UI 簡單、易升級、能表達「重資源 vs 輕樣本」取捨）。
- **雙層容量**：**背包輕**（隨身、小上限 → 逼即時取捨）＋ **撤離倉重**（主容量、靠科技升級）。對應 §7.4「帶什麼、丟什麼」核心張力。

## 13-6. 物種與外交數值（已定版）

### 13-6.A 好感階梯（已定版・5 階）

### 13-6.B 好感增減來源（已定版）

| 方向 | 來源 | 權重 |
|---|---|---|
| **＋（主力）** | 交易/贈禮、完成**託付任務** | 穩定可控的主要好感來源 |

| **−（負面）** | 在物種領域**犯領/盜採**、殺其同胞、**助其敵人** | 行為有後果，好感下降 |

### 13-6.C 互動模型（已定版）

- **遠征初遇**：依物種預設態度 ＋ 首次抉擇定起點。**關鍵規則：玩家首遇的智慧物種（P1 共生種）必定為中立偏友善**——**初期戰鬥難度來自叢林野獸（掠食生物），而非智慧物種**（避免新手一開局就被外交/戰鬥雙重勸退）。
- **互動選項**：戰 / 談 / 交易 / 結盟 / 託付 / 敵對。
- **母船長期經營**：好感達標 → 物種**派使者來訪母船**（交易/給委託）；完成**託付任務** → 推進好感與劇情；好感**跨 run 永久累積**（§10.2）。

### 13-6.D 結盟回報（已定版・遞進制）

中低階只影響敵友識別與基礎交易；回報集中在**友善與結盟**：

- **友善**：解鎖招募、進階交易、託付任務。
- **結盟（最高）**：①**解鎖相容性 → 新星球**（§5/§6，外交直驅探索範圍）②**專屬交易/特產管道**（獨家貨架、特產折扣、稀有資源穩定來源）③**可招募該物種船員 ＋ 奇幻種專屬模塊**（§6 奇幻種）。
- （裂潮防衛戰援軍**不**列入結盟回報，防衛戰維持母船自身佈局為主，§8。）

### 13-6.E 可招募船員池（已定版）

- **每物種 2~3 名可招募**：P1 共生種、P2 遊牧民族、P3 澱臨種各 2~3 名（友善階解鎖招募）。
- **好戰種族叛逃者**：好戰種族**艱難修好**後，出**稀有「叛逃者」船員**（見下）。
- 這些物種船員是 §10.4「30 名船員名單」的一部分；各帶屬性/專長/種族特性/combo 標籤（§10.2），派駐母船模塊。

### 13-6.F 好戰種族外交（已定版・連動結局）

- **可艱難修好、但有代價**：需**極高魂語 ＋ 特殊條件**才能部分緩和；過程中**可能與其他勢力衝突**（修好好戰種族 ↔ 得罪其敵對方）。
- **回報**：修好解鎖**叛逃者稀有船員**與獨特互動。
- **連動 §4 三選一結局**：能否與好戰種族/那股力量和解，呼應**共融**結局路線；純敵對則導向**掌控/否定**路線。

## 13-7. 裂潮母艦防衛戰細節（已定版）

> 細化 §8 玩法骨架的四項：波次、登艦敵類型、砲塔種類、失守倒退量。
> **停泊位×戰場地形對照表**屬 §13-11，本節引用不重複；蜂群智能不升級（§14），挑戰升級來自波次強度與觸發源。

### 13-7.A 觸發與預警（已定版）

- **觸發源**（§8）：時間推移、深入探索、靠近異象帶/傷疤、劇情節點。
- **可預警・有整備窗**：來襲前給**預警 1 個整備回合**，玩家可補強防禦、重排佈局、調整電力網（§5.3/§5.4）。防衛戰＝**佈局 build 驗收**，公平可規劃。

### 13-7.B 波次與勝負（已定版）

- **固定波數・守住即勝**：每場 **3~5 波**遞增，**撐過最後一波＝勝**。
- 非時限制、非守單一核心；結構清晰、有「擋住就贏」成就感。
- 失敗**非 game over**（見 §13-7.E）。

### 13-7.C 登艦敵類型（已定版・3 原型）

| 敵型 | 行為 | 主要剋制 |
|---|---|---|
| **衝鋒群體**（快/弱/多）| 大量小型快速湧入，靠數量淹沒 | 點防攔截器、力場緩衝 |
| **破壞者**（慢/肉/攻模塊）| 高血慢速，專拆模塊/結構 | 玩家親自跨船攔截、雷射砲集火、封艙門阻隔 |
| **投射/飛行體**（遠程）| 從遠處打模塊/角色 | 砲塔、點防攔截器 |

> 裂潮＝半實體異象實體、無意識、蜂群（§14）。**潛行/相位體留 EA 擴充**；停泊位的「隱形危險區」（§6.5）作**環境危險**處理（§13-11），非登艦敵型。

### 13-7.D 防禦模塊戰鬥分工（已定版・維持 §5.7 四種）

| 模塊 | 戰鬥角色 | 對應剋制 |
|---|---|---|
| **砲塔** | 通用自動遠程 | 通用，主清投射/飛行體 |
| **雷射砲** | 高傷單體、耗電高 | 破壞者/重型 |
| **點防攔截器** | 攔截近身小型/投射 | 衝鋒群體、投射體 |
| **力場產生器** | 護盾、吸收波次傷害 | 全體緩衝 |

- **修補無人機庫**：戰時自動修受損模塊。
- **封艙門/氣閘**：阻擋登艦裂潮、分隔船段——**限制破壞者推進路線**，是佈局戰術核心。
- **combo S3**（砲塔＋力場＋雷射砲）＝防禦矩陣（§5.8）。

### 13-7.E 失守倒退量（已定版）

- **模塊損毀需重修**：被打毀模塊受損/停機，需資源重修（綜合工廠/合金，§5.7）。
- **船員受傷**：→ 醫療艙治療（吃時間/居住，§5.6）。
- **已解鎖格數（＝修復進度＝勝利條件線，§5.1）不倒退**。
- > **精化 §8**：原「修復進度倒退」精確為**模塊損毀**；永久勝利線不倒，維持懲罰明確但不挫敗。

### 13-7.F 強度成長軸（已定版・觸發源決定強度）

- **越靠近異象帶/傷疤、越深入探索 → 來襲越強**（非以修船進度為主軸）。
- **設計後果**：形成「**保守求穩 vs 冒進求肥**」風險取捨；玩家行為驅動難度。
- 配合軌道地形（§6.5/§13-11）：當前**戰場地形決定易守/難守**（受襲方向/掩體/隱形危險區），但**不影響產出**（肥險解綁）；強度仍主由觸發源決定。

### 13-7.G 玩家定位（已定版・雙模式可切）

- **親戰模式**：跨船親自打登艦敵（主對破壞者/衝鋒群體），動作手感為主。
- **接管/修復模式**：接管砲塔精準射擊，或跑去修受損模塊、調電力、關封艙門。
- **隨時切換**，對應 §8「跑船 ＋ 砲塔混合」（UI/製作複雜度已知，併入 §13-9 後續細化）。

- **防禦投資**：前哨**砲塔/隱蔽升級**降低波及機率/損失——呼應 §13-7「保守 vs 冒進」取捨。

### 13-9.G 背包 / 撤離介面（已定版）

- **拾取＝自動拾取入背包 ＋ 隨時可開背包管理/丟棄**：走過自動拾取，玩家**任何時候**可開背包整理/丟棄（超重時另彈取捨提示），不打斷探索節奏。
- **背包 / 前哨 / 撤離倉三段**（§13-2.D、§13-10.D）：
  - **背包隨身（輕）**＝ run 層，死亡全損。
  - 走到**前哨**可「存入庫存」＝**世界層永久保留**（死亡不損），求穩選項。
  - 走到**撤離點**把物資載入**撤離倉**（重）＝**仍屬 run 層，須成功撤離才入母船庫存**，死亡則損。
  - 介面同屏顯示三者，強化「送回前哨求穩 vs 賭更深一趟」的撤離張力（§7.4）。

## 13-10. 存檔架構（已定版）

> 把 §9.4（永久 meta vs run 臨時）、§5.6（資源分層）、§13-2.G（戰敗全損/meta 不損）落實成遊戲存檔。沿用原 TRPG 分層 state 思路，改為遊戲存檔格式。

### 13-10.A 三層存檔分層（已定版）

| 層 | 內容 | 死亡時 |
|---|---|---|
| **meta 層** | 母船佈局/修復進度/格子解鎖、科技樹、技能星網節點、永久點數（🔧科技/✨技能/🏆GE）、**母船資源庫存** | **永不損** |
| **世界層** | 星球解鎖/相容性、物種好感/外交協議、劇情・KNOWN/HIDDEN 旗標、**前哨設置與庫存** | **永不損** |
| **run 層** | 當前遠征臨時狀態：角色位置、HP/PSI/氧氣、背包、撤離倉、臨時 build/拾取增益 | **死亡全清** |

> 三層獨立寫檔，損失邊界＝層邊界，與 §13-2.G 完全對齊。

### 13-10.B run 中途存退（已定版・ironman 式）

- **可暗存續玩**：run 層持續自動寫入，玩家可隨時退出、下次從中斷處續玩。
- **死亡即定**：死亡/撤離結果**立即提交**並清 run 層，**不能讀檔逃避**（防 save-scum，保 §2 採集-撤離張力）。

### 13-10.C 存檔時機（已定版）

- **關鍵節點自動存**：出航、撤離、建造、招募、防衛戰結束、進入新區域層等節點自動寫入 meta/世界層；run 層持續自動寫入。**無手動存點**（與 B 防 save-scum 一致）。

### 13-10.D 死亡損失邊界（已定版）

- **背包**（run 層）＝死亡全損。
- **撤離倉**（run 層）＝**須成功撤離才入母船庫存（meta 層）**，死亡則損。
- **前哨庫存**（世界層）＝死亡保留。
- **meta/世界層**＝永不因死亡損失。
- > 設計後果：玩家可在「載入撤離倉賭更深一趟」與「先送回前哨求穩」之間取捨（§13-9.G）。

### 13-10.E 格式與防毀損（已定版）

- **格式＝JSON/字典明文**（Godot Dictionary→JSON）：可讀、好 debug、易做版本遷移；單機 roguelite 不防手改。
- **防毀損＝原子寫入 ＋ 雙備份**：寫臨時檔→原子重命名，保留上一份備援；偵測損壞時回退備份（meta 尤其要）。
- **EA 相容＝版本號 ＋ 遷移腳本**：每層存檔寫 schema 版本號，載入時跨版跑遷移腳本補預設值，**保 meta/世界層不破**（§12 EA 頻繁更新的關鍵保障）。

### 13-10.F 存檔槽 / 雲端（已定版）

- **多存檔槽**：允許多個獨立存檔（不同開場角色/周目並行，§10.4）。
- **無雲端（首發）**：本地多槽優先；**Steam Cloud 同步留 EA 後續**（§12），減輕首發工程量。

## 13-11. 母船移動數值（已定版）

> 補 §6.5 母船移動的數值。**首發聚焦軌道地形系統**；跨星自航留 EA 中期（§12）。本節修訂 §6.5 ①（軌道微移→漂移＋隨機地形＋燃料 reroll＋後期軌道圖直選）。

### 13-11.A 軌道地形系統（前期・首發核心）

- 母船在起始星軌道上**隨時間漂移**（非整備直選位置）。
- **防衛戰準備期**看到當前位置的**防守地形（系統隨機生成）**。
- **燃料 reroll**：可花**中成本燃料**重新隨機生成地形（限制亂 reroll）。
- **後期軌道圖**：🔬研究通訊線・感測升級（§9.2）解鎖「軌道圖」→ 預測並**直接挑選有利地形**（不靠隨機）。
- **位置/地形只影響防衛戰，不影響產出**（肥險解綁）。

### 13-11.B 防守地形特性軸（隨機生成維度）

| 軸 | 變化 | 影響 |
|---|---|---|
| **受襲方向** | 單向 / 雙向 / 環襲 | 砲塔/防禦怎麼擺；單向易守、環襲難守 |
| **掩體/障礙物** | 多 / 中 / 少 | 跑船躲避與登艦敵推進路徑 |
| **隱形危險區** | 有 / 無 | 近異象帶地形有隱形傷害/干擾區（§6.5），需避開或需異感看見 |

### 13-11.C 防守地形對照（範例組合）

| 地形範例 | 受襲方向 | 掩體 | 隱形區 | 守備難度 |
|---|---|---|---|---|
| 穩固軌位 | 單向 | 多 | 無 | 易 |
| 開闊軌位 | 雙向 | 少 | 無 | 中 |
| 紊流軌位 | 環襲 | 中 | 無 | 難 |
| 異象帶緣 | 雙/環 | 少 | 有 | 最難 |

> 玩家可 reroll 求較易地形，或接受隨機應變；後期軌道圖直接挑選。

### 13-11.D 自航燃料經濟（已定版）

- **燃料來源**：副反應爐穩定產（§5.7）。
- **軌道內**：地形 reroll ＝ **中成本固定燃料**（限制亂 reroll）。
- **跨行星航行**：**章節里程碑驅動**（非自由燃料飛）；**首發不做跨星自航**，留 EA 中期（§12）。

### 13-11.E 自航途中事件/里程碑（EA 中期）

- 跨星自航 ＝ **點對點跳躍 ＋ 途中隨機事件**（遇裂潮/遺跡/求救）。
- 目的地 ＝ **章節解鎖的特定點**（傳送到不了的隔離區、**裂潮異象核心＝終局航點**）。
- **首發不實作**，隨跨星自航於 EA 中期上線（§12）。
