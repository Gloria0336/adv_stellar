## 1. 世界觀

### 1.1 核心前提

你的種族派出一艘**大型多功能探索船**前往一個未知星系，已知該星系可能存在智慧生命與各種未知危險。
官方任務僅為**探索與科研**；更深的來訪原因被列為機密，**主角開局並不知情**（見 §2 裏層）。

船在**進入星系前**遭到一股不明威脅——代號**「裂潮」**——重創。主角被**震波擊暈**，
再次醒來已是**數日後**：飛船在失去動力的漂流中，被一顆星球的引力**鎖定於軌道上**（該星球＝起始世界與第一個探索目標）。
此時船體嚴重受損，原本約 30 名船員死亡 27，只剩 **3 人**：你，以及 **2 名重傷者**——
**通訊官**與**科學家**（在休眠艙中強制休眠以避免死亡）。原船滿載可容納約 **200 名船員**，
如今絕大部分功能報廢或損壞（開局隨機決定損壞分布）。

**裂潮並非一次性事件**：它是一種**蜂群式存在**，會**間歇性湧現、攻擊母艦**，可被防守、也可被反擊——
母艦因此承擔**戰鬥角色**（見 §7.3）。其本質與那股星系級力量的關係見 §1.3。

你的任務分三層（見 §2）：修好飛船、進行科研並帶回成果 (GE)、以及一條更深的暗線。

### 1.2 「天外即惡」文化

星系內存在數種智慧物種，對外來者（你）態度各異、**可隨遊戲逐步改善**。
但他們共享一條文化邏輯：**傳送、資源、生命皆源於行星本身，故星球之外的一切皆為威脅／惡。**
這一句話同時解釋了：原住民普遍的排斥、好戰種族的敵意、以及傳送機制為何排斥你（見 §1.4）。
改善關係的本質，就是**讓他們相信你不是天外之威脅**——一項貫穿全程的長期工程。

### 1.3 星系級力量（折衷設定）

擊毀你飛船的，是星系中一個**好戰且具敵意的種族**所引動的「星系級力量」。
他們對其也只是**一知半解**；引動它、把裂潮砸向你們、造成傷亡後，便**失去了掌握**——
裂潮自此在星系中**失控、間歇湧現**（見 §1.1、§7.3）。
這股力量、以及「掌握它的方法」，正是你的種族當初前來此星系的原因之一。

**本質**：它底層是一種**中性、無意識的宇宙底層場**（近似重力場／暗能量的層級），極度危險。
- **裂潮＝它的衍生物**：被引動/擾動時，從空間裂隙湧出一潮**異象實體**（半實體的力量碎片）。這群實體**大體無意識**，是天災般的物理威脅。
- 蜂群智能**不隨遊戲升級**（甦醒弧暫不採用）；難度改由來襲波次強度與進度推進（見 §7.3）。
- 力量本身「是否不只是一個場」屬**後期科研謎團**，但不透過蜂群行為表現。
- 三個深層變異（維裂・靜寂・諧振）皆與這股力量同源，對應對它的三種終極立場（見 §8.1）。

### 1.4 行星傳送機制（與星系級力量同源）

原住民靠星系內某種機制掌握了**行星間傳送**，因此從未發展太空科技（不需要）。
此機制是星系級力量的**「馴化／民用版」**，好戰種族動用的則是其**「原始／武器版」**——同源、不同應用。
傳送機制**對你的種族不友善**：初期你被排斥，只能受限躍遷。
**「提升與傳送機制的相容性」是一條貫穿全程的進度線**（見 §5），靠物種好感、科研解析、量子通訊情報逐步解鎖更多可達星球。

### 1.5 奇幻 × 科幻雙層

| 層 | 內容 |
|---|---|
| 科幻硬層 | 飛船、能源、模塊、合金、資料、傳送工程 |
| 奇幻軟層 (a) | 角色**原生異能 + 變異**（§8.1） |
| 奇幻軟層 (b) | 部分物種天生超常能力；部分星球有「異象帶」（物理異常區） |

### 1.6 控制中樞 AI

你的常駐 AI 夥伴，同時擔任旁白、系統介面與任務給予者。**定位：受損修復型 ＋ 夥伴型**——
它在那次攻擊中也受損（記憶缺口、人格碎裂、偶發故障），隨你修復其核心而逐漸恢復，
人格與能力跟著船一起成長。它與你之間是真正在場的羈絆——在這片虛空裡，你不是一個人。
「修復它」既是系統升級，也是這段關係逐步加深的過程；它的復原本身就是一條情感支線。

## 2. 三層任務目標

| 層 | 目標 | 對應系統 |
|---|---|---|
| 表層 | 修好飛船、回家 | 基地修復・擴充 |
| 中層 | 科研探索、帶回成果 (GE) | 研究科技・量子通訊 |
| 裏層 | 理解／取得那股力量，或阻止好戰種族重奪 | 變異・傳送・外交・終局抉擇 |

> 終局抉擇示意：遠程傳回成果（部分勝利）vs 親自帶船返航（完整 GE）；對那股力量的處置，則由**深層變異三選一**（掌控／否定／共融）對應你的終極立場（見 §8.1）。

## 3. 設計亮點：飛船三位一體

> **飛船 = 基地 = 角色之家 = 勝利條件**

### 3.1 修復 ＝ 擴充（甲板即地形）

完整船體的甲板配置圖**就是你的基地網格**——但它**不是一張均質的空白格**，
而是一張有「地形」的廢墟巨艦。每一格繼承原船的固有特性，所以
**「在哪裡、依什麼順序、復活什麼」本身就是策略**，而非單純填格子。

- 仍可運作的艙室 = 預先放好的**起始模塊**。
- 報廢／封閉的區段 = **鎖住的格子**，用「結構・材料」科技逐塊清理、修復、重建。
- 理論最大值 = 那艘可容納 200 人的完整巨艦（終局超大基地）。
- 你不是在「蓋新基地」，而是在**讀懂這艘廢墟的骨架、決定復活什麼**。

**格位固有特性（tile traits）**

| 格位 | 適合放置 | 風險／代價 |
|---|---|---|
| 船脊・動力幹線 | 反應爐、高耗能模塊（Flux 吞吐高） | 稀少，兵家必爭 |
| 船殼緣（對外） | 感測、武裝、機庫、氣閘（需對外視野/弧度） | 受攻擊時最先被打 |
| 核心內艙（受保護） | 休眠艙、關鍵研究、指揮 | 無對外通道 |
| 破損區 | 便宜、快速佔用 | 不穩定，需先穩固，否則觸發災害事件 |
| 冷卻區 | 放置熱源可避免輻射擴散 | 位置固定 |

**三條空間變數（疊在固有特性之上）**
1. **修復路徑**：只能從仍運作的艙段向外接力清理，無法跳修遠處孤立格 → 擴張有地理與順序。
2. **攻擊幾何**：周邊模塊替核心擋傷 → 權衡「對外存取 vs 受保護」的佈局決策。
3. **場域輻射**：熱／輻射／結構負載從特定模塊向外輻射，放哪裡決定波及範圍。

> 以上是「位置」的第一層深度；再疊上 §7.1 的相鄰 combo（模塊彼此互動），placement 共有**兩層策略**。

### 3.2 起始艙室配置（你列的 ＋ 補入的引擎件）

**可運作（起始模塊）**
- 登陸艙 ×1（含隨船基礎工具組 → 充當早期維修/製造能力）
- 船長室、船員艙 ×3、艦橋（附基礎分析終端，免開局全盲）
- 醫療艙、休眠艙（內含 2 名重傷倖存者：**通訊官**與**科學家**）
- 貨艙（20% → 起始儲存上限，初期逼你取捨）
- **應急能源核心**（低出力，僅夠維持生命維持/醫療/休眠）
- **生命維持系統**（降載 → 人口上限機制，現約 5~8 人，修復後逐步逼近 200）

**報廢／待修（早期目標）**
- **主反應爐**（恢復它＝第一個關鍵目標，解除能源限制）
- **研究/實驗艙**（重建它＝第一個真正的科研階段目標）
- （選配）水耕艙、其餘大量封閉艙段

> 喚醒 2 名休眠倖存者 = 初中期目標：有情感重量，各多 1 名船員，並牽動敘事性解鎖條件——
> **通訊官** → 量子通訊／網識變異；**科學家** → 研究解析效率／變異馴化（他若死亡，相關進度大幅延後）。

### 3.3 母船 ＋ 星球前哨

- **母船**：核心主基地，模塊化與 combo 的主舞台。
- **星球前哨**：設於特定星球的小型採集基地，受該星危險度影響，定期回收資源。

## 4. 核心遊戲循環

```
   星系傳送圖 ──選目標──▶ 星球/地點 ──探索/採集/事件(2d6)──▶ 資源/情報
        ▲                                                      │ 帶回母船
        │ 相容性↑ / 修船↑ / 強化下一趟                            ▼
   去更遠 ◀──招募/外交── 物種 ◀──研究/修復/建造/排列(combo)── 基地建設
```

1. 在傳送圖選一顆（相容性已解鎖的）星球前往。
2. 表面探索／採集／事件，用 2d6 判定。
3. 帶資源回母船：研究科技、推進修船、建造/重排模塊、解析變異。
4. combo 解鎖新產出/buff；與物種外交、招募船員。
5. 相容性與修船進度提升 → 解鎖更遠星球與更深劇情。

## 5. 移動系統：行星傳送（相容性成長線）

- 世界 = 一個星系；每顆星球/天體 = 獨立 region JSON（防 LLM 失憶）。
- 移動靠**原住民傳送機制**，但你初期被排斥，可達範圍受限。
- **相容性 (compatibility)** 是貫穿全程的進度值，提升來源：
  - 物種好感／外交成果
  - 科研解析傳送機制
  - 量子通訊取得的情報／座標
- 相容性↑ → 解鎖更多星球、更低風險、更快傳送。
- 部分星球需特定條件（科技/好感/變異）才可抵達。

## 6. 物種外交系統（核心系統）

態度各異、可改善、且好戰種族是主線威脅——故外交不是附屬，而是**核心系統之一**。

| 維度 | 說明 |
|---|---|
| 態度 | 各物種對你的好感/敵意，受行動與「天外即惡」文化影響 |
| 互動 | 招募為船員、交易、結盟、託付任務、敵對 |
| 影響 | 可去哪（相容性）、可換什麼、能否接近那股力量 |
| 奇幻種 | 部分物種天生超常（精神種、共生種…），提供獨特能力或專屬模塊 |

> 魂語柱與「網識／共生」變異在此價值極高（§8.1）。

船員欄位：屬性／專長（決定可指派模塊）／種族特性／好感／combo 標籤。

## 7. 基地系統（母船 ＋ 前哨）

- **放置兩層策略**：① 格位固有地形（§3.1）② 模塊相鄰 combo（§7.1）。
- 模塊類型：能源、生產・採集、研究、生命・居住、防禦、特殊（融合/共鳴）。

### 7.1 相鄰與 Combo（三層）
1. **相鄰加成**：相鄰特定模塊互增益（如 能源↔生產 +15%）。
2. **組合解鎖**：湊齊一組解鎖新功能（如 反應爐+異象導管+共鳴核心 → 全船能源 +40%）。
3. **互斥干擾**：某些不能相鄰（如 反應爐↔居住 → 船員受輻射 -20%）。

### 7.2 擴充上限（三重限制）
1. 格數：由結構・材料科技 + 船體修復進度決定。
2. 能源：模塊運轉需 Flux，不足則停機。
3. 核心模塊：大型模塊需特定核心。

### 7.3 母艦戰鬥（裂潮防衛）

裂潮蜂群會**間歇來襲**，母艦因此承擔戰鬥角色。此系統把基地佈局直接變成戰術決策：

- **防禦模塊**：力場、砲塔、攔截器、修補無人機；佈於**船殼緣**才有射界（呼應 §3.1 攻擊幾何）。
- **來襲波次**：觸發來源——時間推移、深入探索、靠近異象帶/傷疤、劇情節點；強度隨進度升級。
- **守 vs 反擊**：可純防守消耗，或主動反擊（砲火＋你的操軛/燼能＋船員）取得材料/樣本/情報。
- **佈局即戰術**：周邊擋傷、核心受保護、能源能否撐住戰時負載——§3.1 的格位地形在戰鬥中見真章。
- **失守後果**：模塊損毀、船員受傷、修復進度倒退（而非直接 game over），維持「修復—受創—再修復」的節奏張力。

> 蜂群智能**不隨甦醒升級**（甦醒弧暫不採用）；挑戰升級來自波次強度與進度推進，而非敵人變聰明。

## 8. 雙軌成長系統

### 8.1 技能星網（奇幻・個人）

**起源：原生基本異能 ＋ 變異。** 原生讓你不必以凡體面對巨大未知；變異是「未知力量」，正是科研要解析的對象。
**拓樸：星網輻射 + 五原生柱 + 碰撞處長出變異。** 分支網狀、點數＋條件解鎖。

**中心：覺醒原點（原生核心）** — 異能火種，所有路徑由此向外。

**五原生柱（已知・點數即可解鎖）**

| 原生柱 | 玩法支柱 | 原生基本能力 | 科技交叉 |
|---|---|---|---|
| 識域 | 探索 | 感知・掃描・危機預知 | 感測・通訊 |
| 操軛 | 戰鬥/通用 | 念動・力場・遠端操作 | 結構・材料 |
| 同調 | 科技協同 | 與機械/AI 共鳴・增益模塊 | 推進・能源 |
| 魂語 | 社交/招募 | 共感・心靈感應・跨種溝通 | 感測・通訊 |
| 蛻體 | 生存 | 適應惡劣環境・再生 | 生命・生產 |

**五變異能力（未知・兩柱碰撞處・需科研解析）**

| 變異 | 碰撞 | 效果 | 額外條件 |
|---|---|---|---|
| 觀界 | 識域×操軛 | 預知・機率操弄・重擲骰 | 高階研究解析 |
| 燼能 | 操軛×同調 | 操控熱/輻射/能量場・武裝 | 反應爐重啟 |
| 網識 | 同調×魂語 | 接通 AI/集體意識・量子共鳴 | 量子通訊上線＋喚醒通訊官 |
| 共生 | 魂語×蛻體 | 與異星生命結絆・召喚操使 | 接觸特定物種 |
| 異感 | 蛻體×識域 | 感知異象帶・看見不可見 | 進入異象帶取樣 |

**三深層變異（終局・唯一性・三選一）**

全程只能最終解鎖**其中一個**，代表你對那股力量的終極立場，導向不同結局與重玩價值。
三者皆需把那股力量「解析到底」才能觸及。

| 深層變異 | 立場 | 內容 |
|---|---|---|
| 維裂 | 掌控 | 瞬移・相位・摺疊空間，駕馭它的空間之力（與傳送/躍遷同源，高風險） |
| 靜寂 | 否定 | 壓制異能・製造死域・消音那股力量，拒絕並保護世界 |
| 諧振 | 共融 | 與這股力量共融、理解、繼承，成為橋 |

> 與 §2 裏層任務的終極抉擇直接對應：掌控／否定／共融。三條路互斥不可回頭。

### 8.2 科技樹（科幻・集體・四大類保留）

| 領域 | 解鎖內容 |
|---|---|
| 推進・能源 | 反應爐、躍遷/傳送相容性升級、能源儲存 |
| 結構・材料 | 合金、模塊外殼、擴充上限、**船體修復** |
| 生命・生產 | 生命維持、農圃/合成、自動採集 |
| 感測・通訊 | 星圖解析、研究加速、**量子通訊**、物種通訊 |

> **量子通訊**（感測・通訊）：與母星建立連線，取得情報/劇情/遠程技術支援，並合理化「帶回成果」與終局返航抉擇。
> **研究 = 解析未知**：科技不只是造東西，更是「馴化你身上與星系中的未知」——這把兩棵樹咬合在一起。

### 8.3 雙樹交叉依賴・點數・敘事條件

- **交叉依賴**：變異節點需技能前置 ＋ 科技前置 ＋ 敘事條件三者皆備（範例見「網識」）。
- **點數來源**：經驗、研究、探索里程碑、量子通訊解鎖。
- **敘事條件**：可為事件結果（如通訊官存活與否影響網識解析效率）。
- **節點給予**：新能力／解鎖／條件加成／資源效率（四者皆用）。

### 8.4 「龐大樹」與「2d6 不崩壞」並存

- 樹主要提供**質變**（新能力、解鎖、條件加成、效率），故可無限做大。
- 影響擲骰的**平擺修正值設上限**：核心屬性 +0~+3、情境 +1~+2。
- DC 隨進度提高，相對挑戰恆定。
- 一句話：**樹給你「能做更多事」，而非「擲骰永遠成功」。**

## 9. 判定系統（2d6 + 細修正）

| 結果 | 區間 | 意義 |
|---|---|---|
| 完全成功 | 10+ | 達成，無額外代價 |
| 成功有代價 | 7–9 | 達成，但伴隨損耗/風險/複雜化 |
| 失敗 | 6− | 未達成或惡化 |

- 修正值上限：屬性 +0~+3、情境 +1~+2、特定能力為條件式。
- 優勢/劣勢：3d6 取較高/較低兩顆。
- 擲骰歸屬：GM 報 DC 與修正明細，玩家擲骰回報（跨平台公平一致）；Artifact 版可改 `Math.random()`。
- 戰鬥/採集/社交/研究/駕駛/修理共用同一核心。

## 14. 災變正典「裂潮」（Canon Lock・不可違背）

> 本節為**既定事實**。GM 不得改寫、不得即興補充與此衝突的細節，並須嚴守 §14.3 的 KNOWN/HIDDEN 邊界。
> （建repo 時抽出為 `data/canon/rift_tide.json` + 一條 PROTOCOL 規則。）

### 14.1 裂潮是什麼
- **名稱**：裂潮 (Rift Tide)。
- **構成**：**異象實體**——半實體的力量碎片，自空間裂隙湧出。
- **性質**：那股星系級力量的**衍生物**（非力量本身）；**大體無意識**；如潮汐般湧來、退去、再來。
- **行為**：間歇來襲母艦，可防守、可反擊（§7.3）；**智能不升級**。

### 14.2 逐拍時間線（既定）
1. 進場前：飛船於星系**邊緣外側**做最後減速，一切正常。
2. **T0**：毫無預警，裂潮自空間裂隙湧現，淹沒外緣與前段艙室。
3. 衝擊：27 名船員於外緣/前段短時間內罹難；主角被**震波擊暈**。
4. 退潮：裂潮如潮水般退去（真相：引動方失控）；船失去動力。
5. 漂流：失能的船漂流**數日**。
6. 捕獲：被一顆星球引力鎖定，進入軌道（＝起始世界）。
7. 現在：主角醒來——應急能源、AI 受損重啟、2 名重傷者休眠、27 死、船多數報廢。**遊戲開始**。

### 14.3 KNOWN / HIDDEN（防飄移閘門・最重要）

| 玩家 T0 已知（可當事實呈現） | 隱藏真相（達條件前**不得**揭露） |
|---|---|
| 官方任務＝探索＋科研 | 真實來訪目的：取得那股力量與其掌握法（解鎖：量子通訊母星／解密船載機密／AI 記憶復原） |
| 邊緣外側遭不明威脅「裂潮」無預警重創 | 裂潮是某**星系級力量的衍生物**（解鎖：科學家＋研究解析蜂群樣本／異象帶取樣・異感） |
| 裂潮＝來歷不明的半實體蜂群，淹沒船體後退去 | 該力量被一**好戰種族蓄意引動**、瞄準你船、隨後失控（解鎖：外交/探索接觸該種族 or 量子通訊情報） |
| 27 死；你＋通訊官＋科學家生還（2 人休眠） | 原住民的行星傳送＝**同源力量的馴化版**（解鎖：深入外交/科研） |
| 被震波擊暈，數日後醒於某星球軌道 | 力量「是否不只是一個場」＝後期科研謎團 |
| AI 受損、記憶有缺；裂潮會間歇再襲 | — |

### 14.4 傷亡與遺體
- 27 死於外緣/前段被淹沒的艙室。
- 3 名生還者皆位於受保護的**核心區**（你近艦橋；通訊官與科學家事後被送入醫療→休眠）。
