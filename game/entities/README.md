# entities/ — 實體層（架構決策 3/8）

**各類各自獨立系統 ＋ 共用底層原語**：每類移動實體有自己的行為/決策（清楚分割），
但移動/尋路/感知不重複造輪（避免 solo 重複陷阱）。

## `_core/` 共用原語（所有移動實體共用）
| 檔 | 職責 |
|---|---|
| movement_core.gd | 速度/轉向/基礎位移 |
| navigation_core.gd | 尋路（封裝 NavigationAgent2D）|
| perception_core.gd | 視野/偵測（Area2D 感知）|
| stats_core.gd | HP/狀態值容器 |
| behavior_fsm.gd | 狀態機基類（行為掛載點）|
